❓ Guide d'utilisation

🎮 Lancer une partie ▼

1. Clique sur Créer une partie depuis le menu

2. Choisis ton pseudo et ton emoji

3. Sélectionne un mode de jeu (voir plus bas)

4. Choisis le nombre de questions (5, 10, 15, 20 ou 30)

5. Sélectionne ta playlist (voir plus bas)

6. Clique sur Lancer → un code à 6 chiffres sera généré

7. Partage ce code aux joueurs, ils le tapent pour rejoindre

8. Quand tout le monde est dans le lobby, clique sur Démarrer

💡 Mode Hôte : Coche cette option si tu veux que ton écran affiche les questions en grand (pour projeter sur un écran/TV). Tu ne joueras pas, tu seras le maître du jeu.

💡 Mode Distance : Coche cette option si les joueurs ne sont pas dans la même pièce. Chacun entendra la musique sur son propre téléphone.

🎯 Les modes de jeu ▼

🎯 QCM : 4 choix proposés, trouve le bon titre ou artiste. Fonctionne avec toutes les playlists Deezer.

✍️ Écriture : Tape au clavier le titre et/ou l'artiste. Plus dur mais plus de points ! Fonctionne avec toutes les playlists Deezer.

💿 Titres uniquement : Comme l'écriture mais uniquement les titres. Fonctionne avec toutes les playlists Deezer.

🎉 Soirée : Mix de QCM et d'écriture pour varier les plaisirs. Fonctionne avec toutes les playlists Deezer.

⚠️ Les modes ci-dessous nécessitent des playlists spécifiques :

🎧 Disney Mix : 3 niveaux (Normal → Reverse → Extrait). Fonctionne uniquement avec la playlist Disney.

🎬 Film : Retrouve les films à partir de dialogues et musiques. Fonctionne uniquement avec la playlist Films.

📺 Série : Retrouve les séries à partir de génériques et extraits. Fonctionne uniquement avec la playlist Séries.

🎵 Utiliser une playlist Deezer ▼

Option 1 — Playlist enregistrée :

Dans l'onglet 🎵 Deezer, choisis une playlist dans la liste déroulante. Ce sont les playlists ajoutées par les administrateurs.

Option 2 — Lien Deezer :

Tu peux aussi coller un lien Deezer directement dans le champ URL. Va sur Deezer, trouve une playlist, copie son lien (ex: https://www.deezer.com/fr/playlist/123456789) et colle-le.

💡 Si tu colles un lien, la playlist enregistrée sera automatiquement désélectionnée pour éviter les conflits.

📚 Playlists Films, Séries et Disney ▼

Ces playlists sont spéciales et pré-enregistrées. Pour les utiliser :

1. Va dans l'onglet 📚 Mes Playlists

2. Sélectionne la playlist correspondante (Films, Séries ou Disney)

3. Choisis le mode de jeu adapté :

• Playlist Disney → mode 🎧 Disney Mix

• Playlist Films → mode 🎬 Film

• Playlist Séries → mode 📺 Série

⚠️ Si tu lances un mode spécial avec la mauvaise playlist, les questions ne fonctionneront pas correctement !

🔑 Changer mon mot de passe ▼

1. Depuis le menu, clique sur ⚙️ Mon Compte

2. Entre ton ancien mot de passe

3. Entre ton nouveau mot de passe (2 fois pour confirmer)

4. Clique sur Changer le mot de passe

👥 Mode Équipes ▼

Coche Mode Équipes à la création pour diviser les joueurs en 2 équipes (🔴 Rouge et 🔵 Bleu).

Les joueurs sont automatiquement répartis. Les scores d'équipes s'affichent en fin de partie en plus des scores individuels.
